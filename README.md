# Redberry Bootcamp XI — Course Platform

An online learning platform where users can browse, enroll in, and track educational courses.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- React Hook Form
- Axios
- React Router DOM

## Features

- Authentication (register, login, logout)
- Course catalog with filters, sorting, and pagination
- Course detail with cascading schedule selection
- Enrollment with conflict detection
- Progress tracking and course completion
- Profile management with avatar upload

## Live Demo

https://redberry-bootcamp-platform.netlify.app/

## Run Locally

```bash
npm install
npm run dev
```
